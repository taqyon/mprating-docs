### Атрибуты уникальности

Атрибуты уникальности — это атрибуты, которые обеспечивают уникальность учетной записи для маркетплейса. **Для чего нужны?**

К атрибутам уникальности относятся: 

- User Agent
- Скрытая геолокация
- Уникальный IP Адрес
